## Query Parameters

`ui/voice_bot_app5.py` では、起動時の状態を URL クエリで指定できます。
例:

```
https://avatar.hyper.or.jp:8501/?init=企業・一般向け講座お申込みフォームについて聞かせてください&model=aoki&lang=ja
```

| パラメータ | 意味 |
|-----------|------|
| `init`    | チャット履歴に最初に挿入するユーザー発話 |
| `model`   | `aoki` / `sakaguchi` / `anton` またはそれぞれの `*_model_v1` 名 |
| `lang`    | 応答の言語 (`ja`, `en` など) |

指定すると、アバターと使用言語が固定され、初期メッセージが1回だけ履歴に追加され、そのまま自動で応答が再生されます。
